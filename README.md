## Collaboration

### Adding Collaborators

1. Go to your repository on GitHub: https://github.com/contactuberscool/rKive
2. Click on **Settings** → **Collaborators** → **Add people**
3. Enter your colleague's GitHub username or email
4. They will receive an invitation to collaborate

### Getting Started (For Collaborators)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/contactuberscool/rKive.git
   cd rKive
   ```

2. **Create a new branch for your work:**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make your changes and commit:**
   ```bash
   git add .
   git commit -m "Description of your changes"
   ```

4. **Push your branch to GitHub:**
   ```bash
   git push origin feature/your-feature-name
   ```

5. **Create a Pull Request:**
   - Go to the repository on GitHub
   - Click "Compare & pull request"
   - Add a description of your changes
   - Request review from collaborators

### Workflow Best Practices

- Always create a new branch for each feature or fix
- Use descriptive commit messages
- Pull the latest changes before starting work: `git pull origin main`
- Keep your branch up to date: `git pull origin main` (while on your branch)
- Create pull requests for code review before merging to main
